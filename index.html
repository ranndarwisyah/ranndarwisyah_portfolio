Report Management System (Ongoing)

A comprehensive system developed for the Mitrans (UiTM) Resource Department to streamline report management. The platform allows users to upload, analyze, and store documents such as theses and reports. **Utilizing advanced machine learning models for document classification and object detection**, the system offers a SciSpace-like search function, enabling users to find specific information or entire documents by prompting with keywords or questions. The system also leverages a Large Language Model (LLM) for semantic search and content analysis. It provides direct PDF downloads or directs users to physical retrieval options if the file is unavailable.
